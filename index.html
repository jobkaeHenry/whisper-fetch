Quick Start

Installation

Terminal
npm install @jobkaehenry/whisper-fetch

Basic Usage

index.ts
import { BackgroundPrefetcher } from '@jobkaehenry/whisper-fetch';


const prefetcher = new BackgroundPrefetcher({
  url: 'https://cdn.example.com/large-file.bin',
  allowOnCellular: false,
  respectSaveData: true
});


await prefetcher.start();

React Hook

VideoPlayer.tsx
import { usePrefetcher } from '@jobkaehenry/whisper-fetch/react';


function VideoPlayer() {
  const { progress, status, objectURL } = usePrefetcher({
    url: 'https://cdn.example.com/video.mp4'
  });


  return (
    <div>
      <p>Status: {status} | Progress: {progress}%</p>
      {objectURL && <video src={objectURL} controls />}
    </div>
  );
}

Key Features

Auto Pause/Resume

Automatically pauses downloads when users actively use the network and resumes when idle. This allows efficient prefetching of large files without disrupting user experience.

Range-based Resume

Uses HTTP Range requests to continue interrupted downloads. No need to restart from the beginning after network errors or user pauses.

Server Requirements

  • HTTP Range support (206 Partial Content)
  • ETag/If-Range recommended (for safe resume)

Browser Support

Chrome 114+FirefoxSafariEdge

Chrome 114+ offers best performance with OPFS streaming. Other modern browsers are supported via IndexedDB fallback.
